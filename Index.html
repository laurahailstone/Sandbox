HTML Notes

HTML (Hypertext Markup Language) a standardized system for tagging text files to achieve font, color, graphic, and hyperlink effects on World Wide Web pages.

HTML Elements

An HTML element is an individual component of an HTML document. It's function is to represent the meaning/semantics of the content. Most HTML elements start and end with "tags" <> and . These tags surround and give meaning to the content. For example, this element is a paragraph. It is surrounded by p tags.

Basic HTML Elements

Some tags give a descriptive structure to your HTML which makes it easier to read and help with Search Engine Optimization. Some examples include:

  • Headers
  • Paragraph
  • Image
  • Link
  • Unordered List (like this one!)
  • Ordered Lists
  • List Items

  • Line Break (did you catch that?)
...and many more! There are so many elements! Thankfully, elements can be nested within other elements. For example, the "unordered list" and "list item" elements have been nested within this paragraph!

There is even a tag which allows you to make comments without them displaying with the rest of your content. Look.... See? No, you can't see it. ...but now you can.

HTML5 Elements

HTML5 introduces more description HTML tags which makes it easier to read and help with Search Engine Optimization. Some examples include:

  • main
  • header
  • footer
  • nav
  • video
  • article
Now it can be confusing to know the difference among these structural elements, so here is some help.

Head tag=information ABOUT your page
Body tag =all the contents of an HTML document, such as text, hyperlinks, images, tables, lists, etc.
Main tag=main content of the document. Should be unique to the document and not contain any content that is repeated across documents such as sidebars, navigation links, copyright information, site logos, and search forms.
Footer tag=usually copyright, authorship, contact information, site map, and links back to the top.
